Chapter 4:

Solution Design

This section will describe the software systems, system users
and integrations forming part of the Supergroup IP Voice project.


4.1 Overview:

3CX IP PBX (Internet Protocol Private Branch Exchange), voice solution was implemented. The new phone system now routes and manages calls within the supply chain, dealerships, and call centre sectors through secure internet utilization.


4.2 Outline:

This project was designed to provide a comprehensive plan that will replace the current voice solution of the client.
The outcome of this project is planned to empower employees, agents and managers, with the needed access and training to fully utilize the identified software resources and features. This should provide the client with the ability to operate independently and effectively whilst keeping true to the organizations highly established customer service and operational standards.



4.3 3CX Software Summary:

Features
  • 3CX voice solution will allow for, but is not limited to the following:
  • Call Forwarding
  • Call Queuing
  • Auto Attendance
  • Reliable and Secure
  • Cost Saving
  • Integration with various devices like IP phones and softphones
  • Accessible across different locations
Key Aspects
  • Key aspects used for real-time voice communication includes:
  • VoIP - Voice over Internet Protocol
  • SIP - Session Initiation Protocol
Technology
  • By leveraging the internet protocol (IP), voice packages are transmitted as digital packages.
  • This will allow for easier scalability and flexibility in the network setup.
Components
  • IP PBX Server:
    The 3CX server is the central unit that handles call management and routing.
  • SIP Trunks:
    ICTGlobe provides the virtual phone lines that connect the IP PBX to the public telephone network.
  • 3CX Extensions:
    ICTGlobe configured the extensions used to manage call groups and call flows effectively.
  • IP Phones:
    As part of the original solution proposed, SNOM handsets and desk IP phones was configures, installed and tested on-site.
  • Softphones: 3CX soft clients may be installed on user devices, including mobile phones and computers. For this project, it was anticipated that soft clients will only be installed if, and when required to do so.
